## Claude:

### 1. URDFファイルが既にある

関節間の距離・構造の情報は **URDF（Unified Robot Description Format）** というファイルに全部書かれている。このPC上に既にある。

場所: `/home/techshare/ILkit/xr_teleoperate/assets/g1/g1_body29_hand14.urdf`

右腕の関節チェーン（関節間の距離）を抜き出すと:

```
torso（胴体）
  │
  │ xyz = (0.004, -0.100, 0.248) ← 胴体から右肩までの距離(m)
  ↓
右肩ピッチ
  │ xyz = (0, -0.038, -0.014)
  ↓
右肩ロール
  │ xyz = (0, -0.006, -0.103)
  ↓
右肩ヨー
  │ xyz = (0.016, 0, -0.081)
  ↓
右肘
  │ xyz = (0.100, -0.002, -0.010)
  ↓
右手首ロール
  │ xyz = (0.038, 0, 0)
  ↓
右手首ピッチ
  │ xyz = (0.046, 0, 0)
  ↓
右手首ヨー → ここにグリッパーが付く
```

`xyz`の数値が「親の関節から子の関節までの距離（メートル）」。例えば胴体→右肩は、横に10cm・上に24.8cm離れている、ということ。

### 2. ライブラリは未インストールだが候補はある

URDFを読み込んで関節の計算をしてくれるライブラリが現在このPCに入っていない。候補は以下:

| ライブラリ | 特徴 | 用途 |
|---|---|---|
| **ikpy** | 軽量、pip一発で入る | URDFからキネマティクスチェーンを作れる。「関節角度→手先位置」の計算ができる |
| **pinocchio** | 高機能、Unitree公式でも使用 | 本格的なロボット力学計算。G1設計書でも言及あり |
| **pybullet** | 物理シミュレーター付き | 3D表示で動作確認もできる |

**上田さんへの確認事項**: どれを入れるか（おすすめは **ikpy** → 軽くてすぐ試せる）、または既に別の環境にインストール済みのものがあるか。

## Claude:

### xyzについて

はい、座標。**親の関節から見た子の関節の位置（メートル単位）**を表している。

```
xyz = (0.004, -0.100, 0.248)
       ↑        ↑       ↑
       x        y       z
      前後     左右    上下
    0.4cm   10cm右  24.8cm上

つまり「胴体の中心から、右に10cm・上に24.8cm の位置に右肩がある」
```

### ライブラリ追加調査: pinocchioが既にインストール済みだった

各ライブラリのインストール可否を調べたところ、**pinocchioが既に2つのconda環境にインストール済み**であることが判明した:

| conda環境 | pinocchioバージョン | 用途 |
|---|---|---|
| `tv` | 3.1.0 | xr_teleoperate（テレオペ）で使用中 |
| `unitree_lerobot` | 4.0.0 | ACT学習で使用中 |

さらに、テレオペのコード（`xr_teleoperate/teleop/robot_control/robot_arm_ik.py`）が既にpinocchioでG1のURDFを読み込んでいた。プロジェクトで実際に使われているライブラリなので、新規導入は不要。

### pinocchioで実際に計算できた

`tv`環境のpinocchioでURDFを読み込み、右腕の各関節の位置を計算できた:

```
関節                           前後(m)   左右(m)   上下(m)
───────────────────────────────────────────────────
右肩ピッチ                     -0.000   -0.100    0.292
右肩ロール                      0.000   -0.141    0.289
右肩ヨー                       -0.000   -0.147    0.186
右肘                            0.016   -0.147    0.105
右手首ロール                    0.116   -0.149    0.095
右手首ピッチ                    0.154   -0.149    0.095
右手首ヨー（グリッパー付根）     0.200   -0.149    0.095
```

つまり **「関節角度を入れたら手先の位置が出る」計算が既にできる状態**。

上田さんへの報告としては:「pinocchioが`tv`環境と`unitree_lerobot`環境に既にあるので、新規ライブラリ導入は不要です」で良いと思う。
